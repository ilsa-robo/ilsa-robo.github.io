ILSA: Incremental Learning for Robot Shared Autonomy

In submission

Yiran Tao1, Helen Qiao2, Dan Ding3, Zackory Erickson1,
1CMU, 2, 3,
arXiv Code

Abstract

Recent learning-based shared autonomy methods often require extensive expert demonstrations for training, which can be resource-intensive and difficult to scale. Moreover, these systems typically remain static after deployment, lacking mechanisms for continuous improvement. To address these limitations, we propose an Incrementally Learned Shared Autonomy (ILSA) framework. ILSA begins with pretraining on synthetic demonstrations generated from intuitive rules and continuously refines its performance through repeated user interactions. With a carefully designed training paradigm, ILSA balances learning from new data with retaining existing knowledge, enabling robust improvement after every user interaction, even in the face of limited data generalizability. We validate ILSA through a comprehensive ablation study and a user study with 20 participants, demonstrating its effectiveness and robustness in both quantitative performance and user-reported qualitative metrics.

RoboGen Pipeline


RoboGen is a fully automated pipeline for endless and diverse skill acquisition. RoboGen pipeline consists of 4 stages: A) task proposal, B) scene generation, C) training supervision generation, and D) skill learning with generated information.

Long-horizon task generation and learning


RoboGen is able to propose high-level tasks, generate corresponding environments, decompose the high-level goal into low-level subtasks, and then learn the sub-skills sequentially.
